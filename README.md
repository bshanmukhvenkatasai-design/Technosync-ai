## 🚀 Getting Started

### Prerequisites
* **Node.js** (v18+ recommended)
* **Git**

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/bshanmukhvenkatasai-design/Technosync-ai.git
   cd Technosync-ai
   ```
2. Install dependencies (once dependencies are added):
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
